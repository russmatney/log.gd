# Show Coverage Skill

Display the test coverage report for the Log.gd plugin.

## Steps

1. Look for test coverage reports in the `reports/` directory
2. Find the most recent coverage report (usually in `reports/coverage/`)
3. Parse and display the coverage information in a readable format
4. Highlight areas with low coverage that might need more tests
5. If no coverage report exists, explain how to generate one with gdUnit4

## Coverage Metrics

This skill will show:
- Overall code coverage percentage
- Line coverage
- Branch coverage (if available)
- Files with low coverage
- Uncovered code sections

## Generating Coverage

If coverage reports don't exist, they can typically be generated by running tests with coverage flags enabled in gdUnit4.
